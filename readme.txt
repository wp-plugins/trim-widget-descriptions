== Description ==

Limit the length of the description under each widget on the the widgets page.

== Installation ==

1. Upload `trim-widget-descriptions` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to /wp-admin/widgets.php to change the description length.
